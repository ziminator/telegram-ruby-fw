1. Подключаем rubygems.
2. Используемые гемы:
- json - гем для получения даных в формате json
- net-http-persistent - управляет постоянными подключениями с помощью Net :: HTTP, включая пул потоков для подключения к нескольким хостам.
- sqlite3 - гем для БД sqlite3
- telegram-bot-ruby - основной гем для создания соединения с телеграм ботом.
